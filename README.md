# A 股量价 + 筹码 + 公司质地分析 Skill

> 给 Claude Code 用的 A 股分析技能。输入一个股票代码,输出一份覆盖**量价 / 筹码 / 公司质地 / 政治/地缘风险**的结构化诊断报告。
>
> 不是预测涨跌,而是把"主力意图 / 筹码结构 / 公司类型 / 外部风险"四件事讲清楚,让你自己判断。

---

## 方法论

### 三条核心原则

**1. 量在价先** — 成交量是唯一不滞后的真指标。主动资金(市价单)推动价格,像蒸汽推动火车。价格可以骗线,量很难。所以**所有买卖信号必须由量能确认**。

**2. 筹码换手(守恒)** — 交易即换手,各价位持仓占比形成筹码峰。**筹码守恒**:每日筹码保留率 = (1 - 换手率),主力再怎么对倒、藏仓,真实换手数据藏不住。峰越高 = 该价位持仓越多 = 那里的人越多。下方筹码 = 支撑(套牢盘盼回本不卖),上方筹码 = 压力(套牢盘盼解套抛)。**筹码越集中,市场分歧越小,波动合力越强**。**底仓追踪是反人性信号**:涨 30-50% 底部筹码不动 = 主力还在(要翻倍),高位底仓消失 = 主力出货(见顶)。

**3. 成长 vs 周期** — 这两类公司的估值逻辑完全相反:
- **成长股**:靠需求扩张 + 护城河,利润复利增长,**适合长期持有**(时间是优势)
- **周期股**:利润由供给松紧决定,重资产扩产滞后易过剩暴跌,**只能波段操作**(时间会杀了你)

> **铁律:把周期股当成长股死扛,是 A 股最常见的亏损模式。** 碳酸锂 2022 年 PE 个位数时是顶部,海运 2020 年行业亏损时是买点 — PE 低不等于便宜,看公司类型。

### 第四条:外部风险敞口

**4. 政治/地缘/政策风险** — A 股公司估值不只看财务,还要看外部风险敞口:
- 海外矿山/油气田 → 资源国政局动荡、国有化
- 半导体/AI 芯片 → 美国制裁、实体清单
- 光伏/新能源车/CXO → 政策依赖、补贴退坡、集采降价
- 稀土/钨/锗 → 出口管制(反向受益)
- 白酒/医美 → 消费政策、反腐

**脚本只做"风险敞口识别"**(公司业务是否暴露在某类风险下),**兑现程度需结合最新新闻和企业公告动态核查**。

## 三层分析框架

```
┌─────────────────────────────────────────────────────┐
│  量价(当下)   →   筹码(历史)   →   公司质地(类型)  │
│  + 政治/地缘风险(外部)                              │
└─────────────────────────────────────────────────────┘
```

| 层 | 回答的问题 | 数据来源 |
|----|-----------|----------|
| 量价 | 主力现在在干什么? | 近 120 日 K 线 |
| 筹码 | 历史套牢盘/支撑盘在哪? | 近 120 日成交量分布 |

| 政治风险 | 外部有没有可能一锤打死? | 行业 + 名称关键词 + 新闻核查 |

## 六大矩阵

### 矩阵 1:5 日量价趋势 × 位置语境

**主信号是 5 日趋势**(线性回归斜率,归一化为日均变化率%),不是单日象限。**单日信号噪音太大**(主力可以单日对倒造量,但很难连续 5 日造假)。

|              | 低位          | 高位          |
|--------------|---------------|---------------|
| 量增价涨     | 黄金信号(主力扫货) | 诱多陷阱(对倒出货) |
| 量增价跌     | 见底信号(接恐慌盘) | 逃顶信号(主力出逃) |
| 量缩价涨     | 筹码锁定(主升浪拿稳) | 买盘枯竭(危险) |
| 量缩价跌     | 洗盘低吸(回调缩量) | 阴跌别碰(无承接) |

**关键洞察**:同样"量增价涨",低位是黄金信号,高位是陷阱。**位置语境决定信号含义**。

### 矩阵 2:筹码峰形态(换手率衰减 + 底仓追踪)

**物理基础:筹码守恒** - 每日筹码保留率 = (1 - turnover_t),turnover_t = vol_t / 流通股本。主力再怎么对倒、藏仓,真实换手数据藏不住。无流通股本时降级到固定 decay=0.05。

**三大形态 + 升级信号**:

| 形态 | 升级信号 | 含义 | 操作 |
|------|----------|------|------|
| 低位单峰 + 无量突破(换手<1.5%) | **主升浪信号** | 主力吸筹完毕,等放量易拉升 | 关注放量,准备低吸 |
| 双峰 + 底仓不萎缩(保留率≥50%) | **洗盘信号** | 主力洗盘,底仓未跑 | 区间内高抛低吸 |
| 高位单峰 + 底仓消失(保留率<20%) | **见顶信号** | 主力出货完成 | 立即减仓 |

**底仓追踪(反人性信号)**:
- 涨 30-50% 底部筹码不动 = **不是出货,反而是要翻倍的信号**(散户早止盈了,底仓不动说明主力还在)
- 高位底仓消失 = 主力出货,见顶信号

**ASR / CYQK 辅助指标**:
- **ASR**(活动筹码,±10% 带内占比):下跌中 ASR 高(≥50%) = 筹码锁死,主力控盘
- **CYQK_WIN**(获利比例,当前价下方筹码占比):<40% 套牢多反弹抛压小;>80% 获利多抛压渐增

### 矩阵 3:成长 vs 周期分类

**财务指标优先于行业归类。** 即使行业是"成长赛道"(半导体、新能源),若财务呈现周期特征,按周期股处理。

| 指标 | 真成长 | 伪成长 / 周期 |
|------|--------|---------------|
| ROIC 趋势 | 长期高且稳定(cv<0.3) | 下滑或不稳定(cv>0.4) |
| FCF / 净利润 | ≥0.5,现金流匹配利润 | <0.3,盈利但现金流差 |
| 利润增长 | 持续为正 | 大起大落 |
| 毛利率趋势 | 上升或平稳 | 下降 |
| 营收增长 | 持续正增长 | 波动或下降 |
| 扣非/净利润 | ≥0.9(主业贡献) | <0.7(一次性损益多) |

**PE 陷阱(关键!)**

| 类型 | PE 表现 | 含义 |
|------|---------|------|
| 周期 | PE 极低(<10)+ 利润近期暴增 | ⚠️ **见顶信号**(供给即将过剩) |
| 周期 | PE 极高或为负(行业亏损) | ✅ **潜在买点**(低谷布局) |
| 成长 | 高 PE + ROIC 稳定增长 | 可接受(业绩消化估值) |
| 成长 | 高 PE + ROIC 下滑 | ⚠️ 估值杀风险 |

**行业叙事维度**:AI 算力 / 国产替代 / 机器人 / 创新药 / 高端制造 — 财务周期但有叙事 = "周期(有成长潜力)",波段为主 + 跟踪 Q2/Q3 业绩验证。

**ROE 深度分析(巴菲特选股核心)** - ROE = 净利润÷净资产,资本效率比净利润增速更重要。杜邦分析拆 ROE 为三因素,识别三种模式:

| 模式 | 三因素 | 案例 | 风险 |
|------|--------|------|------|
| 高净利率驱动 | 净利率 >15% | 茅台/爱马仕 | ✅ 安全,定价权强 |
| 高周转驱动 | 周转率 >1.0 | Costco/沃尔玛 | ✅ 稳健,运营效率 |
| 高杠杆驱动 | 权益乘数 >5 | 房企/银行 | ⚠️ 风险大,杠杆依赖 |

**巴菲特三步筛选**:① 5-10 年均 ROE >15% + 单年 ≥12% ② 资产负债率 <50% ③ 经营现金流 ≥ 净利润。三步全过 = 优质标的。

**假高 ROE 识别**(波音案例式警告):
- 杠杆驱动:权益乘数 >5 + 净利率 <10%(ROE 靠杠杆非经营)
- 一次性收益:扣非/NI <0.7(卖资产/补贴撑高利润)
- 回购缩分母:净资产同比下降但净利润未下降(波音举债回购推高 ROE)

**机构研报评估(市场观点 + 外资分歧)** - 数据来自东方财富研报中心,聚合 A 股研报评级/目标价/盈利预测,并识别外资/港资/台资合资券商观点:

| 信号 | 判断 | 操作含义 |
|------|------|----------|
| 评级共识 ≥60% | 共识强 | 主导评级可信,作为参考 |
| 评级共识 40-60% | 共识中 | 分歧存在,谨慎参考 |
| 评级共识 <40% | 分歧大 | 机构观点分裂,警惕 |
| 目标价上涨空间 ≥20% | 空间大 | 机构看多,但需防一致预期陷阱 |
| 目标价上涨空间 5-20% | 空间中 | 中性偏多 |
| 目标价上涨空间 ≤-5% | 已超目标 | 已透支机构预期,警惕 |
| 外资 vs 内资评分差 ≥1.0 | 外资明显更悲观/乐观 | 外资观点更谨慎(或更乐观),需关注外资动向 |
| 机构认可度 | 强(≥10 篇 + 共识强)/ 中(≥5 篇)/ 弱(<5 篇)/ 无覆盖 | 覆盖少 = 市场关注度低,可能存在预期差机会 |

**外资/港资/台资合资券商识别**(关键词匹配,非硬编码 orgCode):
- 国际投行在华合资:高盛高华、瑞银证券、摩根士丹利华鑫、瑞信方正、野村东方国际
- 中资国际子公司:中银国际、招银国际、建银国际、交银国际、海通国际、华泰国际
- 港台资:群益、元大、凯基、第一上海、汇丰前海

**注意**:免费接口只能拿到东财研报中心的中文研报(合资券商发布),**真正的高盛/摩根/瑞银/野村英文原版研报需付费终端**(Wind/Choice/Bloomberg/Refinitiv)。

### 矩阵 4:量价背离判拐点

- **顶背离**:价新高 + 量未新高 → 逃顶
- **底背离**:价新低 + 无量 / 放量滞跌 → 抄底
- 用 5/20 日均量线金叉/死叉过滤确认

### 矩阵 5:主力骗局识别

| 标记 | 含义 | 应对 |
|------|------|------|
| 假突破 | 曾冲阻力位又跌回 | 不追,等站稳 |
| 对倒造量 | 量比>2.5 但涨幅<1% | 警惕主力出货 |
| 算法无量空涨 | 量比<0.7 但涨幅>2% | 警惕诱多 |

### 矩阵 5.5:主力资金流 × 筹码 交叉验证

**核心:筹码形态相似,主力意图可能相反。资金流是确认信号,不是猜测。**

数据源:东方财富 `push2his.eastmoney.com/api/qt/stock/fflow/daykline/get`(主力 = 超大单>100万 + 大单 20-100万,东财标准)。返回 ~121 天日线资金流。

**主力资金信号**(`indicators.capital_flow.signals`):

| 信号 | 判定条件 |
|------|----------|
| 吸筹(强) | 连续≥5日净流入 + 5日均占比≥10% |
| 吸筹(中) | 连续≥3日净流入 + 5日均占比≥5% |
| 派发(强) | 连续≥5日净流出 + 5日均占比≤-10% |
| 派发(中) | 连续≥3日净流出 + 5日均占比≤-5% |
| 中性 | 信号不足 |

**交叉验证矩阵**(`indicators.chip_cross_validation`):

| 短期筹码趋势 | 主力资金净流入 | 主力资金净流出 |
|---|---|---|
| 主峰上移 | ✅ 强吸筹(双确认) | ⚠️ 派发(高位换手出货) |
| 主峰稳定 | 低位吸筹(悄然收集) | 暗中派发(主力撤离) |
| 主峰下移 | 抄底进场(恐慌盘接货) | 弱势阴跌(无承接) |
| 集中度上升 | ✅ 强吸筹(筹码集中) | 派发尾声(散户接盘) |
| 集中度下降 | 派发(筹码分散) | ⚠️ 强派发(主力+散户双撤) |

**数据源限制**:东财对单 IP 限流严格,频繁请求可能被临时封锁。失败时 `available=False`,主流程不阻塞,交叉验证降级为 unavailable。缓存 TTL 4 小时。

### 矩阵 6:政治/地缘/政策风险

| 风险类型 | 命中行业 | 核查方向 |
|----------|----------|----------|
| overseas_resource | 矿业/油气/有色金属 | 资源国政局、海外资产减值公告 |
| sanction | 半导体/AI 芯片/军工 | BIS 实体清单、出口管制新规 |
| policy_dependency | 光伏/新能源车/CXO/教育 | 政策文件、集采中标价、补贴退坡 |
| strategic_resource | 稀土/钨/锑/锗 | 出口管制清单(通常反向受益) |
| consumer_policy | 白酒/医美/高端消费 | 消费税政策、医美监管、批价变化 |

**风险与叙事的反向关系**:制裁 ⇄ 国产替代(被制裁方受损但国产替代方受益);政策依赖(光伏) ⇄ 成长转周期(补贴退坡 + 产能过剩)。**分析时必须明确:公司是受损方还是受益方?**

## 实战五步核对

**铁律:五步全 ✅ 才考虑买入。** 缺一项降仓位或观望。

| 步骤 | 通过条件 |
|------|----------|
| 1. 位置高低 | 120 日分位 ≤ 50%(非高位) |
| 2. 地量测试 | 今日量 < 20 日均量 × 0.5 |
| 3. 量比 1.5-3 倍 | ROVL = 今日量 / 过去 20 日均量,落在 1.5-3.0 |
| 4. 均量线金叉 | 5 日均量线上穿 20 日均量线 |
| 5. 三天站稳突破位 | 连续 3 日收盘 > 近 20 日最高价 |

**量比说明**:ROVL(今日量 / 过去 20 日均量)非行情软件即时量比。20 日基准反映真实筹码活跃度。< 1.5 量能不足(假突破),> 3.0 放量过猛(一日游/对倒)。

## 矩阵 7:半导体行业特殊处理(短期信号优先)

**触发条件**:`fundamental.semiconductor_handling.is_semiconductor = True`

### 为什么半导体需要特殊处理

半导体行业受**周期 + 技术迭代 + 制裁/国产替代叙事**三重影响,与其他行业判断方式不同:

| 维度 | 普通行业 | 半导体行业 |
|------|---------|-----------|
| 财务数据 | 滞后但可信 | 严重滞后,利润波动大(高研发投入期 + 周期低谷) |
| ROE | 长期均值有参考价值 | 低 ROE 不代表差(高研发期常态) |
| PE 陷阱 | 周期低 PE 见顶/高 PE 买点 | 高 PE/亏损 = 成长期常态,PE 陷阱不适用 |
| 巴菲特三步 | 长期稳 + 低负债 + 现金流匹配 | 对半导体过严,会误判所有半导体为"回避" |
| 估值驱动 | 财务 + 行业 + 叙事 | **短期量价 + 主题叙事 + 制裁政策** |

**铁律:把半导体当普通行业用 ROE/PE 陷阱/巴菲特标准判断 = 系统性误判。**

### 决策优先级(从高到低)

1. **5/10/20 日筹码对比**(集中度变化 + 主峰迁移 + CYQK 趋势)- `indicators.short_term_chip`
2. **5/10/20 日量价趋势**(加速度 + 象限一致性 + 强度)- `indicators.short_term_trend`
3. 五步核对(位置/地量/ROVL/金叉/站稳)
4. 量价背离 + 主力骗局(假突破/对倒/无量空涨)
5. 政治风险兑现程度(制裁/EDA 出口管制/实体清单,需查最新新闻)
6. 基本面(ROE/PE/杜邦/巴菲特,仅作参考,不作决策依据)

### 短期筹码对比关键信号

| 信号组合 | 含义 |
|---------|------|
| 短期集中度上升 + CYQK 下降 + 主峰稳定 | **主力吸筹**(看多) |
| 短期集中度上升 + CYQK 上升 + 主峰上移 | **高位派发**(警惕) |
| 短期集中度下降 + CYQK 上升 | **筹码松动 + 获利盘兑现**(看空) |

### 短期量价趋势关键信号

| 信号组合 | 含义 |
|---------|------|
| 加速上涨 + 5 日量增价涨 + 三窗口一致 | **主升浪**(看多,但需防高位) |
| 加速下跌 + 5 日量增价跌 + 三窗口一致 | **主跌浪**(看空) |
| 加速度稳定 + 象限分歧 | **震荡**(观望) |

### 半导体识别规则

通过行业关键词(半导体/集成电路/EDA/封测/晶圆/光刻/功率半导体/MCU/SoC/CIS/IGBT/SiC/GaN/MEMS 等)或公司名称关键词(中芯/华虹/长电/通富/华大九天/北方华创/中微/寒武纪/海光/龙芯/澜起 等)识别,覆盖 A 股主要半导体公司。

### 半导体投资思路(自动改写)

- **思路**:半导体行业:短期量价筹码为主,基本面为辅
- **操作**:核心看 5/10/20 日筹码 + 量价趋势 + 五步核对 + 政治风险兑现,基本面仅作辅助
- **关键风险**:美国制裁(ECCN/EDA/先进制程)、技术迭代跟不上、周期低谷、国产替代叙事破灭
- **关键催化**:国产替代订单兑现(Q2/Q3 营收 + 扣非持续增长)、制裁反向受益、大基金持股变化、AI 算力需求溢出

## 输出报告 9 段

每次分析严格输出 9 段,每段引用具体数值:

1. **当前位置** — 60/120 日分位,高/中/低
2. **量价趋势诊断** — 5 日趋势主信号 + 单日象限辅助 × 位置语境
3. **筹码峰分析** — 峰态 + 主峰位置 + 支撑压力 + 集中度
4. **公司质地判断** - 类型 + ROIC + FCF + 毛利率 + 扣非 + 营收 + 行业叙事 + ROE 深度(杜邦+巴菲特+假高)+ 机构研报共识 + 外资观点 + 内外资分歧 + 半导体特殊处理(短期信号优先)
5. **政治/地缘/政策风险评估** — 风险敞口 + 结合新闻/公告核查 + 评级
6. **PE 陷阱检测** — 周期低 PE 见顶 / 高 PE 买点 / 叙事溢价
7. **量价背离 + 主力骗局** — 顶/底背离 + 假突破/对倒/无量空涨
8. **五步核对表** — 位置/地量/ROVL/金叉/站稳
9. **综合建议** — 投资思路 + 操作 + 仓位 + 止损 + 风险提示

---

## 综合判断要点

- **强买(成长)**:成长股 + 低位 + 低位单峰 + 5 日量增价涨 + 金叉 + 五步全过 + 政治风险低 → 长期持有
- **强买(周期)**:周期股 + 行业亏损/PE 极高 + 底背离 + 5 日低位放量 + 海外资产无地缘风险 → 波段布局
- **关注(周期有成长潜力)**:周期(有成长潜力) + 低位 + 行业叙事 + Q2/Q3 营收+扣非验证 → 波段为主,业绩兑现后转长期持有
- **强卖(周期)**:周期股 + PE 极低 + 利润暴增 + 高位单峰 → 立即逃顶
- **强卖(政治风险兑现)**:海外资产已减值 / 被列入实体清单 / 政策依赖退坡且无对冲 → 立即离场,不等技术面
- **回避**:伪成长(ROIC 下滑 + FCF 差) 或 任一骗局标记为 true 或 扣非/NI<0.7 或 政治风险评级高且已兑现

## 网络层防护栈

A 股数据源(东方财富/sina)经常出现 `RemoteDisconnected`、TLS 指纹拦截、限流等问题。本 skill 内置多层防护(从下到上):

```
浏览器 headers(9 字段,模拟真实 Chrome)
    ↓
请求限流(500ms 间隔 + 0-300ms 随机抖动)
    ↓
Session 连接池(HTTPAdapter,复用 TCP)
    ↓
Retry(连接异常 + 5xx/429,指数退避 backoff_factor=0.5)
    ↓
外层 _retry_with_backoff(1/2/4s + 随机抖动)
    ↓
数据源 fallback(eastmoney → sina K线 / sina hq / 硬编码行业映射)
    ↓
磁盘缓存(日线 4h / 财务摘要 1d / 名称表 7d)
    ↓
akshare 版本检查(<1.10 提示升级)
```

**缓存目录**:`~/.claude/skills/stock-analysis/.cache/`(可随时删除)

**排查步骤**:
1. 检查 VPN/代理 — 某些海外 VPN 节点阻断国内财经接口
2. 升级 akshare:`pip install --upgrade akshare`
3. 清缓存:`rm -rf ~/.claude/skills/stock-analysis/.cache`
4. 跳过基本面:`python3 analyze.py 000001 --no-fundamental`

## 局限性

- **不预测涨跌**:只诊断"主力意图 / 筹码结构 / 公司类型 / 外部风险",决策由你做
- **财务数据滞后**:基于季报/年报,最新季度可能未披露
- **筹码分析是估算**:基于成交量分布推断持仓成本,非真实持仓数据
- **政治风险需手动核查**:脚本只识别敞口,兑现程度需查最新新闻+公告
- **机构研报仅 A 股合资券商**:免费接口只能拿到东方财富研报中心数据(含高盛高华/瑞银/中银国际等合资券商),真正的高盛/摩根/瑞银/野村英文原版研报需付费终端(Wind/Choice/Bloomberg/Refinitiv)
- **主力资金流限流风险**:东方财富 `push2his.eastmoney.com` 对单 IP 限流严格,频繁请求可能被临时封锁(数分钟到数小时)。失败时 `capital_flow.available=False`,交叉验证降级为 unavailable,主流程不阻塞。缓存 TTL 4 小时,命中缓存可避免重复请求
- **不覆盖港股/美股**:仅支持 A 股
- **不覆盖消息面**:突发利好/利空(政策、并购、业绩预告)不在脚本范围
